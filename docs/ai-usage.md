# AI 사용사례

## 1. 사용 목적

과제 구현 속도와 문서화 품질 향상을 위해 AI를 보조 도구로 활용했습니다.

---

## 2. 사용 범위

### 설계 보조
- 도메인 분리 및 패키지 구조 초안 정리
- 트랜잭션 경계/예외 처리 전략 정리
- 출석/보증금 정책 테스트 포인트 정리

### 구현 보조
- 엔티티/리포지토리/서비스/컨트롤러 구조 초안 정리
- 테스트 케이스 아이디어 정리
- 예외 케이스 체크리스트 정리

### 문서화 보조
- README 초안
- docs 문서 구조화 및 정리

---

## 3. 검증 방식

AI 결과를 그대로 사용하지 않고 아래 방식으로 검증했습니다.

- 과제 명세와 대조
- H2 DB 데이터 확인
- Postman 수동 API 테스트
- 단위 테스트 실행 결과 확인
- 실제 실행 흐름 점검

---

## 4. 최종 책임

최종 설계/구현/테스트/문서화 결과에 대한 책임은 제출자인 본인에게 있습니다.
